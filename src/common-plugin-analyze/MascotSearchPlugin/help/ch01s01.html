Mascot Settings

  1. Select [Tools] in the menu bar and then select [Options] from the drop down menu.

    The Options dialog appears.

    • The Options dialog can also be displayed by clicking the icon in the toolbar.

  2. Select [Search Engine] > [Mascot] in the Options dialog.

  3. Enter the URL in [Mascot URL].

    • Enter the user name and the password if necessary.

  4. Click [OK].
