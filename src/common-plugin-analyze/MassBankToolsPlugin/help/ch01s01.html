MassBank Search Results

The MassBank Search Results pane is displayed after searching with Spectrum Search, Peak Search or Peak Difference Search or after double-clicking on a completed job in the MassBank Batch Job List.

There are four types of MassBank search results.

MassBank Spectrum Search Results

Results of Spectrum Search

MassBank Batch Search Results

Results of Batch Search

MassBank Peak Search Results

Results of Peak Search

MassBank Peak Difference Search Results

Results of Peak Difference Search

After closing the MassBank Search Results pane, it can be displayed again at any time.

  1. Select [View] in the menu bar and then select [Results] > [MassBank Search] from the drop down menu.

    The MassBank Search Results pane appears.

Record Details

The MassBank Search Results pane displays hit spectra. Details of a specific hit spectrum can be displayed.

  1. In the MassBank Search Results pane, double-click the spectrum whose details you want to display.

    The MassBank details pane appears. The selected spectrum's ID and title is displayed in this pane's title bar.

Overlapping

You can superimpose the spectrum of a MassBank search result onto a raw data spectrum to compare them.

  1. In the Sample Tree, double-click a spectrum to be displayed.

  2. Click the icon in the MassBank result pane. (Refer to the section called ÅgSearch ResultsÅh.)

    The search result spectrum is superimposed in the spectrum window, drawn with a green line.

Batch Job List

You can view batch search jobs in the MassBank Batch Job List pane.

  1. Select [View] in the menu bar and then select [Results] > [MassBank Batch Job List] from the drop down menu.

    The MassBank Batch Job List pane appears.

    If a row whose job is completed is double-clicked, the MassBank Batch Search Results pane appears and displays results of that search job. (Refer to the section called ÅgSearch ResultsÅh.)
