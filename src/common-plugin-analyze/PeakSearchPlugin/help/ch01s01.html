Peak Filtering

You can search for the peaks in the spectrum of a sample.

  1. Select [Tools] in the menu bar and then select [Filter] > [Peaks] from the drop down menu.

    The Filter Peaks dialog appears.

  2. Enter the settings.

    Stage

    Specify the MS Stage of the spectrum.

    RT

    Specify the RT range of the spectrum.

    m/z

    Enter the m/z range of the peaks to search for.

    Intensity

    Enter the intensity of the peaks to search for.

    Diff.

    Specify the difference in m/z value when searching for a combination of peaks with a given m/z difference.

    Tol.

    Specify the m/z difference tolerance level.

  3. Click [Filter].

    The filtering results appear.
