Exporting Peaks

  1. While a heatmap with peaks detected is displayed, select [File] in the menu bar and then select [Export] > [2D Peaks] from the drop down menu.

    The Save As dialog appears.

  2. Enter the file name and click [Save].

    The cluster and peak information is saved in text format.
