Displaying the Search Results Again at a Later Date

The link to the Mascot search results is saved. The search results can be displayed later.

  1. Select [View] in the menu bar and then select [Results] > [Search Engine] from the drop down menu.

    The Search Engine Results dialog appears.

  2. Select a result item.

    Click the icon to display the result view.

    Click the icon to display the substance view.

Note:

  • Depending on the server storing it, the Mascot search results file may be deleted after a certain period of time.

  • If the Mascot URL setting has been changed since a search was performed, double-clicking on the result item will not display the Mascot search results.

  • Double-clicking on a result item which was imported via the [Import Peaks] tab will not display the Mascot search results; an error dialog will be displayed.
