Chromatogram Filter

You can modify the way chromatograms are displayed.

  1. Select [View] in the menu bar and then select [Chromatogram Filter] from the drop down menu.

  2. Select one of the three options.

    All:

    Chromatograms containing spectra from all MS stages will be displayed.

    MS:

    Chromatograms containing only spectra from MS stage 1 will be displayed.

    MSn:

    Chromatograms containing spectra from all MS stages except MS stage 1 will be displayed.
