Waveform Filter

  1. While a spectrum/chromatogram is displayed, select [Processing] in the menu bar and then select [Basic Data Manipulation] > [Filter Waveform] from the drop down menu.

    The Filter Waveform dialog appears.

    • The Filter Waveform dialog can also be displayed by clicking the icon in the toolbar.

  2. Enter the settings.

    Filter type:

    Low-Pass Filter (Linear)

    This option smooths the curve. The effect is strong. Not appropriate for data with discontinuous intensity.

    High-Pass Filter (Linear)

    This option exaggerates the difference in intensity.

    Gaussian Filter (Linear)

    This option smooths the curve. The effect is slight. Not appropriate for data with discontinuous intensity.

    Rectangular Window Filter

    This option smooths the curve. The effect is strong.

    Triangular Window Filter

    This option smooths the curve. The effect is moderate.

    Gaussian Window Filter

    This option smooths the curve. The effect is slight.

  3. Click [Filter].
