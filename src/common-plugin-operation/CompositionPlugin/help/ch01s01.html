Sample Fusion

  1. Open two samples.

  2. Select [Processing] in the menu bar and then select [Advanced Data Manipulation] > [Sample Fusion] from the drop down menu.

    The Sample Fusion dialog appears.

  3. Select the Control and Treatment samples from the drop down lists.

  4. Select the RT Alignment setting from the drop down list.

  5. Set the Composition parameters.

    Operation:

    Subtraction

    A new sample will be created by subtracting the Control sample from the Treatment sample.

    Merge

    A new sample will be created by adding the Control sample to the Treatment sample.

    Ratio

    A new sample will be created by dividing the Treatment sample by the Control sample.

  6. Click [Process].

    The created sample appears in the Sample Tree as Sample 3.
