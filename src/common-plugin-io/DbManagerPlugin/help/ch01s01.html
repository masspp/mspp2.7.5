Database

  1. Select [File] in the menu bar and then select [Database] from the drop down menu.

    The Database dialog appears.

    • The database specified in this dialog contains all the peak matrix settings and results that appear in the Quantitation dialog.

    • The Database dialog can also be displayed by clicking the icon in the toolbar.

  2. Click [Parameters] to modify the settings.

  3. Click [OK].
