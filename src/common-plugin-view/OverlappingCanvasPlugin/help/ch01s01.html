Modifying the Overlapping Display Mode
Modifying the Overlapping Display Mode

You can modify the Overlapping display mode.

  1. Select [View] in the menu bar and then select [Pane] > [Overlapping Settings] from the drop down menu.

    The Overlapping Settings pane appears.

  2. Change the settings.

    To remove a spectrum/chromatogram,

    1. Select the spectrum/chromatogram to remove.

    2. Click the icon.

    To change the line color,

    1. Select the spectrum/chromatogram whose color you want to change.

    2. Click the icon and select the color.

    To shift at an angle,

    1. Edit the values for [Offset(%)] and [Angle] and click [Update].

      • You can also change the values by clicking the buttons.

    To change the display position of a spectrum/chromatogram,

    1. Select the spectrum/chromatogram to move.

      The spectrum/chromatogram displayed at the front (bottom) is listed at the top in the Overlapping Settings pane.

    2. Click the icon or the icon to move the position.

      When the icon is clicked the selected spectrum/chromatogram moves up in the Overlapping Settings pane and forward (down) in the display.

    To display the maximum peak height of each spectrum/chromatogram as the same,

    1. Check [Auto Y Scaling].

      The Intensity scale is no longer shown.

Overlapping Display Mode

The Overlapping display mode displays multiple spectra or chromatograms in a single view.

  1. Select [View] in the menu bar and then select [Display Mode] > [Overlapping] from the drop down menu.

    • The Overlapping display mode can also be selected by clicking the icon in the toolbar.

  2. In the Sample Tree, double-click the spectrum/chromatogram to be displayed.

  3. In the Sample Tree, double-click the spectrum/chromatogram to be superimposed in the window.

    The selected spectrum/chromatogram is added.

    Repeat Step 3 to add more spectra/chromatograms.
