Stacked Display Mode

The Stacked display mode displays multiple spectra or chromatograms vertically in one window.

  1. Select [View] in the menu bar and then select [Display Mode] > [Stacked] from the drop down menu.

    • The Stacked display mode can also be selected by clicking the icon in the toolbar.

  2. In the Sample Tree, double-click the spectrum/chromatogram to be displayed.

  3. In the Sample Tree, double-click the spectrum/chromatogram to be added to the window.

    The selected spectrum/chromatogram is added at the bottom.

    Repeat Step 3 to add more spectra/chromatograms.

Modifying the Stacked Display Mode

You can modify the Stacked display mode.

  1. Select [Tools] in the menu bar and then select [Options].

    The Options dialog appears.

  2. Select [View] > [Window] > [Stacked] in the Options dialog.

  3. Enter a value for [Window Size] and click [OK].

    • The value is applied to both spectra and chromatograms.
