Modifying the Stacked Display Mode

You can modify the Stacked display mode.

  1. Select [Tools] in the menu bar and then select [Options].

    The Options dialog appears.

  2. Select [View] > [Window] > [Stacked] in the Options dialog.

  3. Enter a value for [Window Size] and click [OK].

    • The value is applied to both spectra and chromatograms.
