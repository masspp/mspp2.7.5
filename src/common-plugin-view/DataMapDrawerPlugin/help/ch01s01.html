Heatmap Settings

  1. Select [Tools] in the menu bar and then select [Options] from the drop down menu.

  2. Select [View] > [Heatmap] in the Options dialog.

  3. Edit the settings.

    Theme:

    You can select the color scheme.

    Grayscale

    The intensity is shown by the strength of the black or gray. (Minimum intensity - or zero in the case of a negative value - to maximum intensity of the displayed data)

    Thermography

    The intensity is shown by the color scheme used in thermography. (Minimum intensity - or zero in the case of a negative value - to maximum intensity of the displayed data)

    Flame

    The intensity is shown by a color scheme ranging from light blue to light yellow. (- maximum absolute intensity to + maximum absolute intensity of the displayed data; i.e., the range is always centred on zero)

    French

    The intensity is shown by a color scheme ranging from blue to red. (- maximum absolute intensity to + maximum absolute intensity of the displayed data; i.e., the range is always centred on zero)

    Deep Sea

    The intensity is shown by a color scheme ranging from black to light blue. (Minimum intensity - or zero in the case of a negative value - to maximum intensity of the displayed data)

    Scale:

    You can select the scale.

    Linear

    The intensity is displayed in linear scale using the color scheme.

    Square

    The intensity is displayed in square scale using the color scheme. Select this option when you want to differentiate the intensity in the middle range.

    Log

    The intensity is displayed in logarithmic scale using the color scheme. Select this option when you want to differentiate the intensity in the low range.

  4. Click [OK].
