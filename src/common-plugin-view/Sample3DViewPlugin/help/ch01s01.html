3D Display Settings

All the data are displayed in 3D, plotting m/z on the X-axis, RT on the Y-axis and intensity on the Z-axis.

3D Rotation

  1. Left-click in the 3D display and drag.

  2. Release the left mouse button at the desired angle.

    • Right-click in the 3D display and select [Reset] to go back to the previous image.

3D Zoom

  1. Press [Shift] key and left-click in the 3D display and drag.

  2. Release the left mouse button at the desired scale.

    • Right-click in the 3D display and select [Reset] to go back to the previous image.

3D Display Settings

  1. Select [Tools] in the menu bar and then select [Options] from the drop down menu.

  2. Select [View] > [3D Display] in the Options dialog.

  3. Enter the settings.

    Intensity Type:

    Relative

    The Z-axis in the 3D View is adjusted to the maximum intensity of the sample.

    Absolute

    You can specify the maximum intensity to be displayed on the Z-axis in the 3D View. Enter the value in the field below.

    Max Intensity:

    Enter a value when you select [Absolute] in [Intensity Type].

    Synchronized Rotation:

    Checking this box activates the synchronization of zoom and rotation of the 3D View of multiple samples.

    Draw Axis:

    Check this box if you want to display the scales on the axes of the 3D View.

  4. Click [OK].
