Starting Mass++

  1. Double-click the (Mass++) icon on the desktop to start Mass++.

    • You can also start Mass++ by clicking [Start] and then selecting [All Programs] > [Mass++] > [Mass++].

Exiting Mass++

  1. Select [File] in the menu bar and then select [Exit].
