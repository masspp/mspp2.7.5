Displaying the Peaks

  1. Select [Processing] in the menu bar and then select [Quantitation] > [Show Peak Matrix] from the drop down menu.

    The Quantitation dialog appears.

  2. Select the peak matrix from the drop down list.

  3. Click the (Group Plot)/ (Overlap Plot) toggle buttons to select the plots to be displayed.

  4. Double-click the peak in the Quantitation dialog.

    The Group Plot and/or the Overlap Plot appears.

    Overlap Plot

Creating a Peak Matrix

  1. Select [Processing] in the menu bar and then select [Quantitation] > [Create Peak Matrix] from the drop down menu.

    The wizard for creating MassBank records starts.

  2. Enter the settings in the Peak Matrix Name page.

    1. Enter a peak matrix name.

    2. Click [Next >].

      • Click [Skip] to save the peak matrix settings that have been entered. To resume peak matrix creation, refer to ???.

  3. Enter the settings in the Group Settings page.

    1. Click the icon.

    2. Enter the group name and click [Add]

      • Click the color button to change the group color if necessary.

    3. Click the icon and select samples to be included in the group.

    4. Repeat steps A - C to set other groups.

      • To change the group settings (name and color), select the group and click the icon.

      • To delete the group, select the group and click the icon.

    5. Select the sample to be used as the standard from the drop down list.

    6. Click [Next >].

  4. Enter the settings in the Normalization page.

    1. Select [Executing normalization] and enter the settings for normalization.

      • Select [Not executing normalization] if normalization is not necessary.

    2. Click [Next >].

  5. Enter the settings in the RT Alignment page.

    1. Select [Executing RT Alignment] and enter the settings for RT alignment.

      • Select [Not executing RT Alignment] if RT alignment is not necessary.

    2. Click [Next >].

  6. Enter the settings in the Peak Positions Determination page.

    1. Select the peak positions determination method from the drop down list.

    2. Select the peak values calculation method from the drop down list.

    3. Select the filling missing values method from the drop down list.

    4. Click [Next >].

  7. Enter the settings in the Peak Analysis page.

    1. Select [Executing peak scaling].

      • Select [Not executing peak scaling] if peak scaling is not necessary.

    2. Click the icon.

    3. Select the scaling and click [OK].

    4. Select [Executing peak analysis].

      • Select [Not executing peak analysis] if peak analysis is not necessary.

    5. Click the icon.

    6. Select the analysis method and click [OK].

    7. Click [Next >].

  8. Enter the settings in the Others page (Comparative Identification).

    1. Select [Executing identification] and enter the settings for comparative identification.

      • Select [Not executing identification] if comparative identification is not necessary.

    2. Click [Next >]

  9. Enter the settings in the Others page (Merge Matrix Peaks).

    1. Enter the RT tolerance and the m/z tolerance.

    2. Click [Next >]

  10. Confirm the entered values and click [Finish].
