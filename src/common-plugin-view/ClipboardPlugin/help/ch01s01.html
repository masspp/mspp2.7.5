Clipboard

  1. Select [Edit] in the menu bar and then select [Copy as] from the drop down menu.

  2. Select one of the three options.

    Text:

    The list of m/z and intensity values is copied to the clipboard as text data, when a spectrum is displayed.

    The list of RT and intensity values is copied to the clipboard as text data, when a chromatogram is displayed.

    Bitmap:

    The bitmap image of a spectrum/chromatogram or heatmap is copied to the clipboard.

    Metafile:

    The image and property information of a spectrum/chromatogram or heatmap are copied to the clipboard.
